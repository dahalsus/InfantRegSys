Hey Fellow User!

IPAN's Infant Program Online Registration and Evaluation System is a web application that allows automated services for registration and administration of infant participants. The system also lets users to view recent program offerings, sign up to join those programs, and also get notifications associated to the future events. The system also allows administrators to make new groups and new offerings; also share event information within existing program groups.

-------------------------------------------------------------

Logged in members can check their profile using 'My Profile'. If you already have an account 'Sign In'. New ones can create an account using 'Sign Up' button.
